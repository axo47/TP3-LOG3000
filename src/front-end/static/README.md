# 1 But
Le module `static` sert à regrouper les ressources statiques du front-end comme le style de l’interface utilisateur.
# 2 Les principaux fichiers qu’il contient et leurs responsabilités.
style.css : définit l’apparence visuelle de la calculatrice. 
# 3 Hypothèse et dépendance
Hypothèse : aucune

Dépendance : ce dossier est utilisé comme dossier statique par Flask pour la configuration de l’application.

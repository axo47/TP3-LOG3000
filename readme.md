# Un guide d’installation clair (étape par étape).
Voici les étapes un à un afin de pouvoir lancer l'application 

## 1 
Créer un venv à partir de la ligne de commande suivante 
python -m venv venv  pour windows 
python3 -m venv venv pour linux/macos 


## 2 
Rentrer dans le fichier venv à l'aide de la ligne de commande

Pour macos
source /venv/bin/activate 

Pour Windows



### 3 
Installer les différentes dépendance avec la ligne de commande suivante 
pip install -r requirements.txt 

## 4 
Execute la ligne de code suivant : 
cd src

## 5
Une fois que cela est fait, nous pouvons executer la ligne de code suivante
python app.py

## 6 
Une fois l'application construite : tu peux l'executer comme une calculatrice normal.
